# 如何使用

```js
// 初始化日历 
    var date = '2020-6';
    var date_arr = date.split('-');
    // 如果日历已经存在了,记得销毁掉,要不然就会出现多个日历在后面追加.
    if(record.calendar){
        record.calendar.destructor();
    }
    record.calendar = new dhx.Calendar(	
       "calendar", 
       {
           // 日历宽度
           width: "400", 
           css: "dhx_widget--bordered",
           // 控制日历中不能点击
           disabledDates: function(date) {
           	return date.getMonth() != new Date().getMonth();
           },
           // 初始化选中的日期
           value: new Date(),
           // 获取选中日期的格式
           dateFormat:"%Y-%m-%d",
           // 显示选中日期的万年历
           date: new Date(date_arr[0],date_arr[1]-1,1),
           // 只显示当前月份
           thisMonthOnly: true,
       },
    );
// 日历的事件        
    record.calendar.events.on('ModeChange', function () {
        var val = record.calendar.getValue();  // 获取选中的日期 '2020-06-01'
    });
// 如何根据打卡情况,显示日历下面的点点颜色:
    // 根据后端返回的日历日期,给日历添加 class="2020-06-01",这里需要修改下载好的 calendar.js( 函数: Calendar.prototype._drawCalendar line5428) 源码.
    var spanClass = item.date.format("yyyy-MM-dd")
    return dom_1.el("div.dhx_calendar-day", {
        class: item.css,
        _date: item.date,
        tabIndex: 1,
        ".innerHTML": item.day + '<span class="redpoint '+ spanClass +'"></span>'
    });
// 修改 calendar.js( 函数: Calendar.prototype._drawHeader line5554) 源码.可以去掉日历的头部
```

# html2canvas

html2canvas 库的工作原理并不是真正的'截图',而是读取网页上的目标 DOM 节点的信息来绘制canvas,所以它并不支持所有的 css 属性(详细信息),而且期望使用的图片跟当前域名同源，不过官方也提供了一些方法来解决跨域图片的加载问题。

# Installing

npm install html2canvas

import html2canvas from 'html2canvas';

# Usage

* 引入 `html2canvas`库以后,拿到目标 `Dom` 调用一下 `html2canvas方法`就能生成 canvas 对象了
* 由于我们的目标是生成图片,所以还需要再调用 `canvas.toDataURL()`方法生成<img>标签的可以用数据.
html2canvas(document.body).then(function(canvas) {
    var myImg = document.getElementById('myImg');
    myImg.setAttribute('src',canvas.toDataURL());
});
